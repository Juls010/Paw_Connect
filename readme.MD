# 🐾 PawConnect API - Gestión de Adopciones

Proyecto desarrollado con **Node.js**, **Express** y **MongoDB (Mongoose)** para la gestión integral de una protectora de animales.

---

## Requisitos del Proyecto
- **Modelos:** Animal, User y Adoption (Relación 1:N).
- **Validaciones:** Mensajes de error personalizados y validadores de tipo (String, Enum, Number, Boolean, Date).
- **CRUD:** Operaciones completas para cada modelo.
- **Lógica de Negocio:** Actualización automática de disponibilidad del animal al crear/borrar adopciones.

---

## Instrucciones de Instalación

1. **Descomprimir el archivo** y abrir una terminal en la carpeta raíz.
2. **Instalar dependencias:**
   ```bash
   npm install
   ```
3. **Iniciar el servidor:**
    ```bash
   node index.js
   ```
